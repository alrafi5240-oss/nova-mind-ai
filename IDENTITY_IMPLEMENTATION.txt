Chat Request Flow:
┌─────────────────────────────────────────────────────────────┐
│                   User Message                              │
│  ("Who created you?" / "তোমাকে কে বানিয়েছে?")               │
└──────────────────────────┬──────────────────────────────────┘
                           ▼
┌─────────────────────────────────────────────────────────────┐
│                  Detect Language                            │

└──────────────────────────┬──────────────────────────────────┘
                           ▼
┌─────────────────────────────────────────────────────────────┐
│              Check Creator Question?                        │
│  (Match against 18 predefined patterns)                     │
└───┬───────────────────────────────────────┬─────────────────┘
    │                                       │
    │ YES (Creator Question)                │ NO (Normal Question)
    │                                       │
    ▼                                       ▼
┌──────────────────────────┐    ┌──────────────────────────────┐
│  Get Predefined Response │    │   Get OpenAI Response        │
│  (No API call needed) ⚡   │    │   (Calls OpenAI API) ⏱       │
│  <100ms                  │    │   500-2000ms                 │
│                          │    │                              │
│  Languages:              │    │  Includes:                   │
│  - English: ✓            │    │  - System prompt             │
│  - Bangla: ✓             │    │  - Conversation history      │
│                          │    │  - User message              │
└────────┬─────────────────┘    └──────────┬───────────────────┘
         │                                 │
         └─────────────────┬───────────────┘
                           ▼
┌─────────────────────────────────────────────────────────────┐
│              Store in Conversation Memory                   │
│  (Both types stored for context)                            │
└──────────────────────────┬──────────────────────────────────┘
                           ▼

│                 Return JSON Response                        │

│    "conversation_id": "..."                                 │
│  }                                                          │
└─────────────────────────────────────────────────────────────┘
